A React wrapper for the [GISTDA Sphere Map API](https://sphere.gistda.or.th/). Build interactive maps of Thailand with full TypeScript support.

  <Marker position={{ lon: 98.3923, lat: 7.8804 }} title="Phuket" />
</MapDemo>

## Features

- Declarative React components for maps, markers, polygons, and more
- Full TypeScript support with exported types
- Hooks for search, routing, tags, and map controls
- Built-in and custom layer support
- Overlay collection management

## Quick Install

```bash
npm install gistda-sphere-react
```
