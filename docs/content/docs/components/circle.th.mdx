```tsx
  lineWidth={2}
/>
```

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `center` | `Location` | Yes | - | พิกัดศูนย์กลางของวงกลม |
| `radius` | `number` | Yes | - | รัศมีของวงกลมในหน่วยองศา |
| `ref` | `Ref<CircleRef>` | No | - | Ref สำหรับการควบคุมผ่านโค้ด (Imperative control) |
| `title` | `string` | No | - | ข้อความ Tooltip เมื่อโฮเวอร์ |
| `detail` | `string` | No | - | ข้อความ Pop-up เมื่อคลิก |
| `popup` | `PopupOptions` | No | - | การกำหนดค่าป๊อปอัป |
| `visibleRange` | `Range` | No | All zoom levels | ระยะการซูมที่มองเห็นได้ |
| `lineWidth` | `number` | No | `3` | ความกว้างของเส้นขอบเป็นพิกเซล |
| `lineColor` | `string` | No | Preset color | สีเส้นขอบ |
| `fillColor` | `string` | No | Preset color | สีเติม |
| `lineStyle` | `"Solid" \| "Dashed" \| "Dot"` | No | `"Solid"` | รูปแบบเส้นขอบ |
| `clickable` | `boolean` | No | `false` | แสดงเคอร์เซอร์แบบตัวชี้ |
| `draggable` | `boolean` | No | `false` | อนุญาตให้ลากได้ |
| `zIndex` | `number` | No | `0` | ลำดับการซ้อน |
| `onClick` | `(circle: SphereCircle) => void` | No | - | ฟังก์ชันที่จะทำงานเมื่อคลิก |
| `onDrag` | `(circle: SphereCircle) => void` | No | - | ฟังก์ชันที่จะทำงานเมื่อกำลังลาก |
| `onDrop` | `(circle: SphereCircle) => void` | No | - | ฟังก์ชันที่จะทำงานเมื่อปล่อย (สิ้นสุดการลาก) |
