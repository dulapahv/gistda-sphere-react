```tsx
<Polyline
  positions={[
    { lon: 100.4, lat: 13.72 },
    { lon: 100.5, lat: 13.8 },
    { lon: 100.6, lat: 13.72 },
  ]}
  lineColor="blue"
  lineWidth={3}
/>
```

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `positions` | `Location[]` | Yes (min 2) | - | อาร์เรย์ของจุด |
| `ref` | `Ref<PolylineRef>` | No | - | Ref สำหรับการควบคุมผ่านโค้ด (Imperative control) |
| `title` | `string` | No | - | ข้อความ Tooltip เมื่อโฮเวอร์ |
| `detail` | `string` | No | - | ข้อความ Pop-up เมื่อคลิก |
| `label` | `string` | No | - | ข้อความกำกับตรงกลางเส้น |
| `labelOptions` | `MarkerOptions` | No | - | ตัวเลือกของ Marker สำหรับข้อความกำกับ |
| `popup` | `PopupOptions` | No | - | การกำหนดค่าป๊อปอัป |
| `visibleRange` | `Range` | No | All zoom levels | ระยะการซูมที่มองเห็นได้ |
| `lineWidth` | `number` | No | `3` | ความกว้างของเส้นเป็นพิกเซล |
| `lineColor` | `string` | No | Preset color | สีเส้น |
| `lineStyle` | `"Solid" \| "Dashed" \| "Dot"` | No | `"Solid"` | รูปแบบเส้น |
| `pivot` | `Location` | No | Center | จุดหมุน |
| `clickable` | `boolean` | No | `false` | แสดงเคอร์เซอร์แบบตัวชี้ |
| `draggable` | `boolean` | No | `false` | อนุญาตให้ลากได้ |
| `pointer` | `boolean` | No | `false` | แสดงตัวชี้การลาก |
| `zIndex` | `number` | No | `0` | ลำดับการซ้อน |
| `onClick` | `(polyline: SpherePolyline) => void` | No | - | ฟังก์ชันที่จะทำงานเมื่อคลิก |
| `onDrag` | `(polyline: SpherePolyline) => void` | No | - | ฟังก์ชันที่จะทำงานเมื่อกำลังลาก |
| `onDrop` | `(polyline: SpherePolyline) => void` | No | - | ฟังก์ชันที่จะทำงานเมื่อปล่อย (สิ้นสุดการลาก) |
