title: Ref APIs
description: การควบคุมแบบ imperative ผ่าน React refs
---

คอมโพเนนต์โอเวอร์เลย์ทั้งหมดรองรับ React refs สำหรับการควบคุมแบบ imperative

## SphereMapRef

```tsx
const mapRef = useRef<SphereMapRef>(null);

<SphereMap ref={mapRef} style={{ height: '500px' }} />

mapRef.current?.setZoom(15);
mapRef.current?.goTo({ center: { lon: 100.5, lat: 13.75 }, zoom: 12 });
```

| Method | Parameters | Description |
|--------|------------|-------------|
| `getMap` | `()` | ดึง map instance ที่อยู่เบื้องหลัง |
| `setZoom` | `(zoom, animate?)` | ตั้งค่าระดับการซูม |
| `setCenter` | `(location, animate?)` | ตั้งค่าจุดกึ่งกลางแผนที่ |
| `setBound` | `(bound, options?)` | ปรับแผนที่ให้พอดีกับขอบเขต |
| `goTo` | `(target, animate?)` | เคลื่อนที่ไปยังเป้าหมายพร้อมแอนิเมชัน |
| `setRotate` | `(angle, animate?)` | ตั้งค่าการหมุน |
| `setPitch` | `(angle)` | ตั้งค่าความชัน |
| `setFilter` | `(filter)` | ใช้ตัวกรองสี |
| `resize` | `()` | ปรับขนาดให้พอดีกับคอนเทนเนอร์ |
| `repaint` | `()` | บังคับวาดใหม่ |
