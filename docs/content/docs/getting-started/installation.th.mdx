## ติดตั้งแพ็กเกจ

```bash tab="npm"
npm install gistda-sphere-react
```

```bash tab="pnpm"
pnpm add gistda-sphere-react
```

```bash tab="yarn"
yarn add gistda-sphere-react
```

## Peer Dependencies

ไลบรารีนี้ต้องใช้งานร่วมกับ React เวอร์ชัน 19 ขึ้นไป:

```json
{
  "peerDependencies": {
    "react": ">=19.0.0",
    "react-dom": ">=19.0.0"
  }
}
```

## ขั้นตอนถัดไป

เมื่อติดตั้งเสร็จแล้ว คุณต้องมี [API Key](/docs/getting-started/api-key) จาก GISTDA เพื่อใช้งานแผนที่ จากนั้นทำตามคู่มือ[เริ่มต้นใช้งานด่วน](/docs/getting-started/quick-start)เพื่อแสดงผลแผนที่แรกของคุณ
