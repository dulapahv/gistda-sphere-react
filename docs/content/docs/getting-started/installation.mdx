## Install the package

```bash tab="npm"
npm install gistda-sphere-react
```

```bash tab="pnpm"
pnpm add gistda-sphere-react
```

```bash tab="yarn"
yarn add gistda-sphere-react
```

## Peer dependencies

The library requires React 19 or later:

```json
{
  "peerDependencies": {
    "react": ">=19.0.0",
    "react-dom": ">=19.0.0"
  }
}
```

## Next steps

Once installed, you need an [API key](/docs/getting-started/api-key) from GISTDA to use the map. Then follow the [Quick Start](/docs/getting-started/quick-start) guide to render your first map.
