React wrapper สำหรับ [GISTDA Sphere Map API](https://sphere.gistda.or.th/) สร้างแผนที่ประเทศไทยแบบโต้ตอบได้ พร้อมรองรับ TypeScript อย่างเต็มรูปแบบ

## คุณสมบัติเด่น

- คอมโพเนนต์ React แบบ Declarative สำหรับแผนที่ หมุด รูปหลายเหลี่ยม และอื่น ๆ
- รองรับ TypeScript เต็มรูปแบบพร้อม Export Types
- Hooks สำหรับการค้นหา การหาเส้นทาง แท็ก และการควบคุมแผนที่
- รองรับเลเยอร์ในตัวและเลเยอร์แบบกำหนดเอง
- จัดการคอลเลกชันโอเวอร์เลย์

## ติดตั้งด่วน

```bash
npm install gistda-sphere-react
```
